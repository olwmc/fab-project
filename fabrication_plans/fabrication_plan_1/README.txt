Wang Tile Laser-Cut Puzzle
==========================

Files in this archive:
  tiles.svg    laser cut these (red = cut through, blue = engrave)
  paint_key.svg   reference for painting each edge color

Instructions:
  1. Laser cut: tiles.svg
  2. Assemble the puzzle: Work through the sheet files in
     order (sheet_01, sheet_02, …). Within each sheet, pieces
     fill left-to-right, top-to-bottom (row-wise). Lay them
     out in that sequence to reconstruct the original pattern.
     Tabs and slots are color-coded so each piece only fits
     in the correct orientation.
  3. Paint each edge triangle according to paint_key.svg.
     The engraved patterns identify which color goes where.
